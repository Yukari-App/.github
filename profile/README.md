<h1 align="center"><b>Yukari</b></h1>

<h4 align="center">
    <b>Yukari is a modular, open-source manga reader for your Windows desktop.  
    Its community-driven plugin ecosystem allows you to seamlessly add new manga sources.</b>
</h4>
